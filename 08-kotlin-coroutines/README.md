# Kotlin Coroutines & Flow

Modern async programming with Kotlin coroutines and reactive streams.

## Features

- Coroutines for async
- Flow for reactive streams
- Structured concurrency
- Error handling
- Parallel execution
- Flow operators

## Kotlin Features

- Coroutines and suspend functions
- Flow API
- Extension functions
- Data classes
- Null safety
- Smart casts

## Running

```bash
kotlinc CoroutineDemo.kt -include-runtime -d demo.jar
java -jar demo.jar
```
